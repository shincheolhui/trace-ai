# 보안 정책: 비밀번호 규칙

## 비밀번호 정책

### 최소 요구사항

1. **길이**: 비밀번호는 최소 8자 이상이어야 합니다.
2. **복잡성**: 다음 중 최소 2가지 이상을 포함해야 합니다:
   - 대문자 (A-Z)
   - 소문자 (a-z)
   - 숫자 (0-9)
   - 특수문자 (!@#$%^&*)

### 금지 사항

- 사용자 이름이나 계정 ID와 동일한 비밀번호 사용 금지
- 연속된 문자나 숫자 (예: "123456", "abcdef") 사용 금지
- 반복된 문자 (예: "aaaaaa") 사용 금지
- 일반적인 단어나 사전 단어 사용 금지

### 비밀번호 예시

**적합한 비밀번호:**
- `P@ssw0rd123`
- `MyS3cure!Pass`
- `Tr@ce-AI-2024`

**부적합한 비밀번호:**
- `mypass123` (특수문자 없음, 일반 단어 포함)
- `password` (너무 짧고 단순함)
- `12345678` (숫자만 포함)

### 정책 위반 시 조치

비밀번호 정책을 위반한 경우:
1. 비밀번호 변경 요구
2. 보안 교육 실시
3. 반복 위반 시 계정 잠금
