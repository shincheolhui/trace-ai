# 장애 사례: Redis 연결 오류

## 사례 개요

**발생 일시**: 2024-01-15 14:30
**장애 유형**: Redis 연결 실패
**영향 범위**: 전체 애플리케이션 서버

## 증상

```
ConnectionRefusedError: [Errno 111] Connection refused
Redis 서버에 연결할 수 없습니다.
```

## 원인 분석

### 주요 원인

1. **Redis 서버 미기동**
   - Redis 서비스가 실행되지 않음
   - 시스템 재부팅 후 자동 시작 설정 누락

2. **네트워크 문제**
   - 방화벽 규칙으로 인한 포트 차단
   - Redis 서버와 애플리케이션 서버 간 네트워크 단절

3. **설정 오류**
   - 잘못된 호스트 주소 또는 포트 번호
   - 인증 정보 불일치

## 해결 방법

### 즉시 조치

1. Redis 서비스 상태 확인
   ```bash
   systemctl status redis
   ```

2. Redis 서비스 시작
   ```bash
   systemctl start redis
   ```

3. 연결 테스트
   ```bash
   redis-cli ping
   ```

### 예방 조치

1. Redis 자동 시작 설정
   ```bash
   systemctl enable redis
   ```

2. 헬스체크 모니터링 설정
3. 방화벽 규칙 점검

## 관련 문서

- Redis 운영 매뉴얼
- 네트워크 구성도
- 장애 대응 절차
